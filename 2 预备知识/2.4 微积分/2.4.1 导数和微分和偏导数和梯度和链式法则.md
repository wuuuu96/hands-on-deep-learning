导数和微分和偏导数和梯度和链式法则

1.利用极限来求导

*求f(x)=3x**2+2*x在x=2处的导数*

```python


def f(x):
  return 3*x**2 + 2*x

def lim(f,x,h):
  return (f(x+h) - f(x))/h

h = 0.1

for i in range(5):
  print(f"h = {h:.5f} lim = {lim(f,2,h):.5f}")
  h *= 0.1

```

2.利用自动求导(atuograd)

*求f(x)=x**3+1/x在x=1处的值和导数值*
import torch

x = torch.tensor(1.0,requires_grad=True)

f = x**3 - 1/x

f.backward()

f.item() #函数值
x.grad.item() #导数值
